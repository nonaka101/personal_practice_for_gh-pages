## Tips

- 複数行入力できるボックスは、ダブルクリック（タップ）で入力量に応じて拡張します
- リストアイテム入力時は、特定キーによる操作が可能です
  - `Enter` で新規リストアイテムが挿入されます
  - （入力ボックスが空の場合）`Backspace` でリストアイテムが削除されます
