このページでは、下記のSVGを使用している。
viewbox が 縦横 40(px) に対し、text font-size="50" に指定しているのは、意図的である。 本来 文字列の上下に余白が入ってしまうが、 svg によってそれが切り取られる。 これにより、ボタンとして使用するとき、フォントの上下余白がタッチ範囲に影響を及ぼすことがなくなる。 


	<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 40 40">
		<text x="50%" y="50%" font-size="50" style="dominant-baseline:central;text-anchor:middle;">
			⚀
		</text>
	</svg>
